# Code of Conduct

## Our Standard

This project aims to be a respectful, constructive place for discussion and collaboration.

Examples of behavior that help keep the project healthy:

- Being respectful in disagreement
- Giving actionable technical feedback
- Assuming good intent while reviewing contributions
- Staying focused on the code, docs, and user impact

Examples of unacceptable behavior:

- Harassment or personal attacks
- Trolling, intimidation, or sustained disruption
- Publishing private information without permission
- Abusive, discriminatory, or hostile language

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer at `contact[at]srujangurram[dot]me`.
